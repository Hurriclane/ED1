BG reflection

        Prior to this project I had very limited experience with coding, and I was never particularly skilled in coding so this project was a bit of a struggle. Starting off the bat when I started coding for just the color of my background the code did not work. When I went to the index.html in a web browser I would get a blank white screen, upon further inspection there were two very small spelling mistakes in the code, once fixed the background worked. I had created two GIFs and ended up choosing my second one for my background. I drew 8 different images on a tablet and then uploaded them to photoshop and put them in sequence. The end result was a GIF of the moon phases with clouds moving in front. When adding the GIF to the code it didn’t initially work despite using the proper method. I ended up having to use Mrs. Rileys code and altering it to fit my design. I ended with a dark indigo background with my GIF on top. During the design review I received a lot of positive feedback, and one of the wonders that stood out to me was what would it look like if it was faster and smoother. To do that I would have to make more images and add them to the sequence. Otherwise there were not too many changes that were suggested to my background.
